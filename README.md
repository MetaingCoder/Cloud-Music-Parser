# 云音乐全功能解析工具 (Cloud Music Parser)

一款基于原生 Web 技术栈开发的极简音乐外链解析工具，旨在为用户提供最纯粹、高效的音频资源获取体验。

## 🚀 项目特性

- **极致极简**：采用“去盒子化”设计理念，卡片式布局配合网易红经典配色。
- **智能识别**：自动过滤干扰字符，支持纯歌曲 ID 或 官方分享链接的快速识别。
- **无损导向**：直接调用官方音频流接口，获取当前可用的最高音质（320kbps MP3）。
- **零依赖性**：不使用任何复杂的第三方解析 API，仅通过逻辑拼接，确保解析永不失效。
- **响应式适配**：针对 PC 及移动端进行像素级优化，确保在不同屏幕下均有良好的交互感。

## 🛠️ 技术实现

- **布局**：Flexbox 响应式布局 + 绝对/相对定位嵌套。
- **视觉**：CSS3 变量、毛玻璃效果（Backdrop Filter）及高阶贝塞尔曲线动画。
- **逻辑**：原生 JavaScript 配合正则表达式（Regex）提取 ID 核心参数。
- **交互**：非阻塞式异步加载，解析结果动态呈现，不破坏原始盒子结构。

## 📖 使用说明

1. 在输入框内粘贴网易云音乐的歌曲 ID（如：`1407551413`）或分享链接。
2. 点击右侧的解析按钮（或按下回车键）。
3. 解析成功后，点击下方出现的下载卡片即可获取音频外链。
